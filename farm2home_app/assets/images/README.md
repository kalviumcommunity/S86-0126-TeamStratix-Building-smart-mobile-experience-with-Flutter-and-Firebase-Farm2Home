# Image Assets

This folder contains image assets for the Farm2Home app.

## Recommended Images

Add the following images to this folder:

1. **logo.png** (200x200px) - App logo
   - Use your team/company logo or Farm2Home branding
   
2. **banner.png** (1200x400px) - Homepage banner
   - Hero image for the main screen
   - Could be a farm landscape or agriculture-themed image
   
3. **farm-hero.png** (1080x720px) - Farm/agricultural hero image
   - Could show vegetables, farming, harvest scene

4. **profile-placeholder.png** (200x200px) - User profile default
   - Default avatar for user profiles

5. **welcome-screen.png** (1080x1920px) - Welcome screen background
   - Fullscreen background for onboarding

## How to Add Images

1. Download or create your images (PNG, JPG, or WebP format)
2. Place them in this folder (assets/images/)
3. They are automatically registered in pubspec.yaml
4. Reference them in code as: `Image.asset('assets/images/logo.png')`

## Image Best Practices

- Use PNG for graphics with transparency
- Use JPG for photographs
- Use WebP for better compression (modern browsers/apps)
- Optimize images to reduce app size
- Use appropriate resolution (2x, 3x variants for different devices)
